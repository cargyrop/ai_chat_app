Local Models / Ollama

Run AI models entirely offline on your machine. No API key required.

Setup:
1. Install Ollama from https://ollama.com
2. Pull a model: ollama pull llama3.2 or ollama pull mistral
3. Ollama runs automatically in the background
4. Click Refresh Models below — your local models will appear
